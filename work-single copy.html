Project Overview

Info

I recently co-developed a trading application, utilizing AWS, Tailwind, and Firebase. I designed a secure backend system that ensures smooth account creation, login, and error handling while syncing both databases, AWS and Firebase.

In addition, I took charge of the application's design, utilizing Tailwind to create a sleek, modern interface that is both mobile-friendly and adaptable, complete with a dynamic dark mode feature that effortlessly transitions across devices.

As part of this project, I also created an innovative chat feature that is integrated with Firebase and Firestore, providing additional experience on top of trading feature for users. Ongoing enhancements to the mobile responsiveness and user input form are top priorities, and I work closely with our Scrum Master to continually optimize and improve the overall user experience.
