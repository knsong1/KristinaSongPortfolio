Project Overview

Info

In the Blog project, I showcased my expertise in building full-stack web applications using React, Redux, Redux Toolkit, and Sequelize. I designed and implemented a robust data model using PostgreSQL, allowing users to post their thoughts for the day and search, delete, and favorite their posts. To manage state and optimize performance, I leveraged the powerful Redux Toolkit, implementing best practices such as memoization and reselect. I also ensured data integrity and efficient querying using Sequelize, a popular Object-Relational Mapping (ORM) library. Throughout the project, I followed best practices such as modular code organization, code reuse, and testing, resulting in a scalable, maintainable, and high-performing application. Overall, this project demonstrated my ability to deliver sophisticated full-stack applications using modern web technologies and best practices.
