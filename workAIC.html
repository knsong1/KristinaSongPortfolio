Project Ovierview

Info

This project was designed and implemented using a combination of essential web technologies, including JavaScript, HTML, and CSS. By integrating essential front-end technologies with my expertise in JavaScript, I successfully delivered a fully functional search engine that allowed users to explore the rich and diverse collection of the Art Institute of Chicago effortlessly. To achieve this, I designed an intuitive and user-friendly interface that ensured ease of use and accessibility for all visitors.

One of the key functionalities of the search engine was to make asynchronous API calls to the Art Institute of Chicago's API using the modern Fetch API. This allowed users to retrieve art-related data efficiently and display it dynamically on the web page. Leveraging the power of the Fetch API along with the `async/await` pattern, I developed a utility function to handle these asynchronous requests, ensuring smooth and responsive user interactions.
